## 문제 내용

$L$개의 `(`와 $R$개의 `)`로 만들 수 있는 문자열 중에서 올바른 괄호 문자열인 부분 문자열을 최대한 많이 갖는 문자열을 찾아서, 그러한 부분 문자열의 개수를 출력하세요.

### 입력

첫 번째 줄에는 테스트 케이스의 개수 $T$가 주어집니다. ($1 \le T \le 100$)

각 테스트 케이스에 대해, $L$과 $R$이 한 줄에 주어집니다. ($0 \le L, R \le 10^5$, $1 \le L + R \le 10^5$)

### 출력

각 테스트 케이스에 대해, `Case #{테스트 케이스 번호}: {문제의 정답}`의 형식으로 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

작은 수에 대해 브루트포스를 해 보면, $k = \min(L, R)$일 때 `()`를 $k$번 반복한 것을 부분 문자열로 가질 때 최대임을 알 수 있습니다. 이때의 답은 $\frac{k(k+1)}{2}$입니다.

이것이 답이 되는 이유는, `(`를 $+1$로, `)`를 $-1$로 바꿔서 누적 합에서 같은 수를 최대한 많이 만드는 방법을 생각해보면 됩니다.
