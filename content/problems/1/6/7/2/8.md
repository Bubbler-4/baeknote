## 문제 내용

$N$개의 다트가 꽂힌 위치가 주어질 때, 다트의 점수의 총합을 구하세요. 각 다트의 점수는 원점에서의 거리 $d$에 따라 다음과 같이 결정됩니다.

* $d \le 10$: 10점
* $10 < d \le 30$: 9점
* $30 < d \le 50$: 8점
* $50 < d \le 70$: 7점
* $70 < d \le 90$: 6점
* $90 < d \le 110$: 5점
* $110 < d \le 130$: 4점
* $130 < d \le 150$: 3점
* $150 < d \le 170$: 2점
* $170 < d \le 190$: 1점
* $190 < d$: 0점

### 입력

첫 번째 줄에 다트의 개수 $N$이 주어집니다. ($1 \le N \le 100$)

다음 $N$줄에는 각각의 다트의 $x$좌표와 $y$좌표가 주어집니다. 각 좌표는 정수입니다. ($-300 \le x, y \le 300$)

### 출력

점수의 총합을 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

주어진 그대로 구현하면 됩니다. 하나의 다트에 대해 점수가 여러 번 계산되지 않도록 주의합니다.
