## 문제 내용

Vasya는 팀을 꾸려서 어떤 프로그래밍 대회에 나가려고 합니다. 이 대회에서는 인터랙티브 문제 $n$개와 일반 문제 $m$개가 출제됩니다.

Vasya를 제외한 팀원들은 인터랙티브 문제와 일반 문제 중 한 종류만을 풀 수 있고, 인터랙티브 문제를 하나 푸는 데는 $p$, 일반 문제를 하나 푸는 데는 $q$의 시간이 소요됩니다.
Vasya는 두 종류의 문제를 모두 풀 수 있고, 역시 인터랙티브 문제를 하나 푸는 데는 $p$, 일반 문제 하나에는 $q$의 시간이 소요됩니다.

대회 진행 시간 $t$ 이내에 모든 문제를 풀기 위해 Vasya가 꾸려야 하는 팀의 총 팀원 수의 최솟값을 구하세요.

### 입력

첫 번째 줄에는 테스트 케이스의 개수 $T$가 주어집니다. ($1 \le T \le 1000$)

각 테스트 케이스에 대해, 다섯 개의 양의 정수 값 $n$, $m$, $p$, $q$, $t$가 한 줄에 주어집니다. ($1 \le n, m, p, q, t \le 10\;000$, $p, q \le t$)

### 출력

각 테스트 케이스에 대해, 문제의 정답을 한 줄에 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

인터랙티브 담당 팀원이 시간 내에 풀 수 있는 인터랙티브 문제의 수는 $K_i = \left\lfloor \frac{t}{p} \right\rfloor$,
일반 문제 담당이 풀 수 있는 일반 문제의 수는 $K_r = \left\lfloor \frac{t}{q} \right\rfloor$입니다.

인터랙티브 담당 팀원의 수 $x_i$를 고정하면, Vasya가 풀어야 하는 인터랙티브 문제의 수는 $V_i = \max(0, p - K_i x_i)$입니다. 이 문제들을 시간 내에 못 푸는 경우는 제외합니다.
이제 Vasya가 남은 시간 동안 풀 수 있는 일반 문제의 수를 구할 수 있고, 거기서 남는 일반 문제를 모두 풀기 위해 필요한 일반 문제 담당의 수 $x_r$을 $K_r$을 가지고 구할 수 있습니다.

이제 가능한 모든 $x_i$에 대해 $1 + x_i + x_r$의 최솟값을 구해서 출력하면 됩니다.
