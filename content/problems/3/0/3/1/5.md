## 문제 내용

좌표평면 위에 $k$개의 점이 주어집니다. $k$개의 점 중 하나를 골라서, 나머지 $k-1$개의 점들과의 거리의 평균이 가장 작도록 하려고 합니다.

그렇게 점을 선택했을 때 $k-1$개의 점들과의 거리의 평균을 출력하세요.

### 입력

첫 줄에 $k$가 주어집니다. ($2 \le k \le 1000$)

다음 $k$줄에 걸쳐서 각 점의 $x$좌표와 $y$좌표가 한 줄에 주어집니다. ($-1000 \le x, y \le 1000$, 좌표의 값은 정수)

모든 점의 위치는 서로 다릅니다.

### 출력

문제의 정답을 출력합니다. 정답과 절대 또는 상대 오차가 $10^{-6}$ 이하이면 정답으로 인정됩니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

$k$가 크지 않으므로 브루트포스를 하면 됩니다.
