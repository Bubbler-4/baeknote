## 문제 내용

음이 아닌 정수로 이루어진 길이 $N$의 수열 $a_1, a_2, \cdots, a_N$과 음이 아닌 정수 $K$가 주어집니다. 당신은 다음의 조작을 원하는 만큼 할 수 있습니다.

* 두 인덱스 $i$와 $j$를 선택합니다. $a_i$와 $a_j$가 $|(a_i \oplus a_j) - (a_i + a_j)| \le K$를 충족하면, $a_i$와 $a_j$의 값을 서로 바꿉니다.

당신이 조작을 끝마쳤을 때 나올 수 있는 서로 다른 수열의 개수를 구하세요.

### 입력

첫 번째 줄에 $N$과 $K$가 주어집니다. ($1 \le N \le 100\;000$, $0 \le K < 2^{19}$)

두 번째 줄에 $a_1, a_2, \cdots, a_N$의 값이 순서대로 주어집니다. ($0 \le a_i < 2^{18}$)

### 출력

문제의 정답을 $1\;000\;000\;007$로 나눈 나머지를 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

먼저, 주어진 수식에서 $(a_i \oplus a_j) - (a_i + a_j)$는 $-2(a_i \operatorname{AND} a_j)$와 같습니다. 따라서 주어진 조건을 $a_i \operatorname{AND} a_j \le \frac{K}{2}$로 바꿀 수 있습니다.
편의상 $k = \left\lfloor \frac{K}{2} \right\rfloor$라고 합시다.

또한, 서로 교환할 수 있는 두 값을 이은 그래프를 생각하면, 하나의 연결 요소 내에서는 충분한 조작으로 항상 원하는 순서로 만들 수 있습니다.
따라서, 주어진 그래프에서 각 연결 요소에 속한 값들을 모두 알 수 있으면, 각 연결 요소 내에서 중복값이 있는 순열의 수를 multinomial로 계산하여 모두 곱하면 답이 됩니다.

연결 요소들은 다음과 같이 분할 정복 + disjoint set으로 찾을 수 있습니다.

* 어떠한 값의 집합 $S$ 내에서 AND가 $k$ 이하인 수의 쌍은 다음과 같이 구할 수 있습니다.
  * $\max(k, \max(S))$의 비트 길이를 $l$이라고 합시다.
  * $S$의 값들을 $l$번 비트가 0인 것의 집합 $S_0$과 1인 것의 집합 $S_1$로 나누고, $S_1$에 속한 모든 값에서 $l$번 비트를 제거합니다.
  * $k$의 $l$번 비트가 0이면, 다음의 두 경우로 재귀합니다.
    * $S_0$ 내에서 AND가 $k$ 이하인 수의 쌍
    * 값 하나를 $S_0$에서 뽑고, 다른 하나를 $S_1$에서 뽑았을 때, 그 사이에서 AND가 $k$ 이하인 수의 쌍
  * $k$의 $l$번 비트가 1이면, 다음과 같이 처리할 수 있습니다.
    * $S_0$ 내에서 임의의 두 값을 뽑은 것과, $S_0$에서 하나, $S_1$에서 하나를 뽑은 것은 모두 조건을 충족합니다. 이들은 모두 하나의 연결 요소에 포함되므로, $|S_0| + |S_1|$번의 union으로 처리할 수 있습니다.
    * $S_1$에서 두 값을 뽑은 경우로 재귀합니다. 이때 $k$에서 $l$번 비트를 제거합니다.

* 두 집합 $S$와 $T$에서 값을 하나씩 뽑았을 때 AND가 $k$ 이하인 수의 쌍은 다음과 같이 구할 수 있습니다.
  * 동일하게 $l$을 정의하고, $S_0$, $S_1$, $T_0$, $T_1$을 정의합니다.
  * $k$의 $l$번 비트가 0이면, 다음의 두 경우로 재귀합니다.
    * $S_0$과 $T_0 \cup T_1$에서 하나씩 뽑았을 때 AND가 $k$ 이하인 쌍
    * $S_1$과 $T_0$에서 하나씩 뽑았을 때 AND가 $k$ 이하인 쌍
  * $k$의 $l$번 비트가 1이면, 다음과 같이 처리합니다.
    * $S_0$과 $T_0 \cup T_1$에서 하나씩 뽑은 경우, $S_1$과 $T_0$에서 하나씩 뽑은 경우는 모두 조건을 충족합니다. 이때, 각각의 경우의 그래프는 이분 그래프가 되므로,
      **양쪽 집합이 모두 비어있지 않을 때만** 두 집합에 속한 모든 값들을 union해 줍니다.
    * $S_1$과 $T_1$에서 하나씩 뽑은 경우로 재귀하고, $k$에서 $l$번째 비트를 제거합니다.

union을 상수 시간으로 볼 때, 각 재귀는 크기를 반으로 줄여서 최대 두 번 재귀하고 추가로 선형 시간을 사용하므로, 전체 과정의 시간 복잡도는 $\mathcal{O}(N \log N)$입니다.
