## 문제 내용

다음과 같은 형식의 코드가 주어집니다.

```
for (int i = 0; i < n; i++) {
    if (i == x_1) i = y_1;
    if (i == x_2) i = y_2;
    ...
    if (i == x_k) i = y_k;
    work (i);
}
```

`i`값의 초기화, 비교 연산, `i`를 업데이트하는 연산, 그리고 `work(i)`를 모두 하나의 연산으로 셀 때, 주어진 코드의 연산 횟수를 구하세요.

### 입력

위의 형식에서 $n$, $x_i$, $y_i$들만 정수로 치환된 형태의 코드가 주어집니다. 예제를 참조하세요. ($0 \le k \le 10^5$, $0 \le x_i, y_i < n \le 10^9$)

### 출력

주어진 코드의 연산 횟수를 출력합니다. 주어진 코드가 종료되지 않는 코드라면 `-1`을 대신 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

먼저, if문들을 모두 통과했을 때 어떤 값이 어떤 값으로 바뀌고 총 몇 번의 연산을 하는지를 구합니다. if문들을 역순으로 보면서 해시맵이나 트리맵을 써서 현재까지의 상태를 관리하고,
새로운 $x_i, y_i$ 쌍에 대해서는 $y_i$가 맵에 이미 있는지에 따라서 $x_i$가 들어왔을 때의 결과를 계산해서 추가해 주면 됩니다.

이제 맵에 없는 값들은 if문을 그냥 모두 통과하므로, `i`가 맵에 있는 값으로 바뀔 때까지의 연산량은 `i`의 다음 변화량과 줄 수의 곱으로 구할 수 있습니다.
맵에 있는 값을 만나면 그 값에 대한 연산량을 더하고, 새로운 값으로 바꿉니다. 이 과정을 더 이상 맵에 있는 수를 만나지 않을 때까지 반복합니다.
만약 같은 수를 두 번 만났다면 사이클이 존재하는 것이므로 `-1`을 출력하고 종료합니다.
