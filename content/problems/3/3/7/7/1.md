## 문제 내용

알파벳 소문자로 이루어진 길이 $N$의 문자열 $S = s_1 s_2 s_3 \cdots s_N$이 주어집니다. $1 \le i < j < k \le N$에 대해, $s_i \neq s_j \wedge s_j = s_k$이면 $(i, j, k)$를 moo triple이라고 합시다.
그리고, 이러한 moo triple의 가치는 $(j-i)(k-j)$로 정의합니다.

이때, 다음의 쿼리를 $Q$개 처리하세요.

* `l r`: $l \le i < j < k \le r$인 moo triple의 가치 중에서 가장 큰 값을 출력합니다. 그러한 moo triple이 없으면 `-1`을 대신 출력합니다.

### 입력

첫 번째 줄에 $N$과 $Q$가 주어집니다. ($3 \le N \le 100\;000$, $1 \le Q \le 30\;000$)

두 번째 줄에 문자열 $S$가 주어집니다.

다음 줄부터 $Q$줄에 걸쳐서, 각 줄에 쿼리가 하나씩 주어집니다. ($1 \le l \le r \le N$, $r - l + 1 \ge 3$)

### 출력

각각의 쿼리에 대한 답을 한 줄에 하나씩 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

각 쿼리는 다음과 같이 처리할 수 있습니다.

* 먼저 각 알파벳 $c$에 대해, $s_i = c$인 $i$들의 목록 $A_c$와 $s_i \neq c$인 $i$들의 목록 $B_c$를 만들어 둡니다.
* 쿼리 $(l, r)$이 주어졌을 때, 각 알파벳 $c$에 대해 다음과 같이 $s_j = c$인 moo triple의 최대 가치를 구할 수 있습니다.
  * 범위 내에서 가장 왼쪽에 있는 $s_i \neq c$인 $i$와 가장 오른쪽에 있는 $s_k = c$인 $k$를 구합니다. 이는 각각 $B_c$와 $A_c$에 대한 이분 탐색으로 구할 수 있습니다.
  * 이제 $s_j = c, i < j < k$인 $j$ 중에서 $(j - i)(k - j)$가 최대인 $j$를 구해야 합니다. $j$는 $i$와 $k$의 중점에 가까울수록 이득이므로,
    $A_c$를 그 중점을 기준으로 두 부분으로 나눈 뒤에 각각에서 이분 탐색을 하여 중점에 가장 가까운 $j$를 두 개 구하고, 그 중 최대를 선택합니다.

시간 복잡도는 알파벳의 개수를 $\sigma$라고 할 때 $\mathcal{O}(Q \sigma \log N)$입니다.
