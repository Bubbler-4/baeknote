## 문제 내용

길이 $\ell$의 트랙이 있습니다. 카트는 트랙의 시작점부터 끝점까지 1 m/s의 속도로 이동하며, 당신은 카트 이상의 속도를 낼 수 있지만 카트를 추월할 수는 없습니다.

트랙 위에 $n$개의 상품 판매대가 있으며, $i$번째 판매대는 트랙의 시작점에서 $d_i$만큼 떨어진 위치에 있고, 그곳에서 상품을 받으려면 $w_i$만큼의 시간을 그 자리에서 기다려야 합니다.

모든 상품을 받고 트랙의 끝점에 카트와 동시에 골인하려면 최소한 어느 정도의 속도를 내야 하는지 구하세요.

### 입력

첫 줄에는 $n$과 $\ell$이 주어집니다. ($1 \le n \le 10^5$, $2 \le l \le 10^8$)

두 번째 줄에는 $n$개의 상품 판매대의 위치 $d_1, d_2, \cdots, d_n$, 세 번째 줄에는 각 상품을 받는 데 걸리는 시간 $w_1, w_2, \cdots, w_n$이 주어집니다. ($0 < d_1 < d_2 < \cdots < d_n < \ell$, $0 \le w_i \le 10^8$)

### 출력

문제의 조건을 충족하는 최소 속도 $v$를 출력합니다. 이 값이 정답과 상대 오차 $10^{-4}$ 이내에 있으면 정답으로 인정됩니다. 어떤 $v$에 대해서도 카트와 동시에 골인할 수 없거나, 최소 속도가 $3 \times 10^8$을 초과한다면
`IMPOSSIBLE`을 대신 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

먼저 이 문제의 결정 문제 버전인 {{< baeknote 34292 >}}를 먼저 해결합니다. 이 코드를 이용하여 먼저 $v = 3 \times 10^8$일 때 가능한지 판별하고, 그렇지 않다면 `IMPOSSIBLE`을 출력하고 종료합니다.

가능하다면, $v < 1.0$이면 불가능하므로 이 두 개의 값을 이용해 $v$에 대한 실수 이분 탐색을 수행하면 됩니다.
