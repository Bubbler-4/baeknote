## 문제 내용

생략

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

이 문제를 풀기 위해서는 다음의 관찰이 필요합니다.

* 사과가 홀수 개 남았을 때 자신의 턴인 사람은, 자신이 가져갈 사과를 제외하고 남은 쌍들을 잘 옮기면 반드시 각 쌍의 최댓값을 모두 획득할 수 있으나, 이를 초과하여 획득할 수는 없습니다.
  상대는 각 쌍의 최솟값만을 획득하게 됩니다.

이는 다음과 같이 설명할 수 있습니다. 편의상 각 쌍의 사과들에 a, b, c...의 알파벳 이름을 붙이고, 각 쌍 내에서 큰 것을 대문자, 작은 것을 소문자로 쓰겠습니다. 그러면 선공은 다음과 같이 사과를 배열할 수 있습니다.
(X는 선공이 가져간 사과를 의미합니다.)

```none
X a B c D ... d C b A
```

그러면 후공은 `a`를 가져가야 하고, 자리를 바꿀 수 있는 쌍들은 모두 둘 다 대문자이거나 둘 다 소문자입니다. 다시 말해서, 후공은 대문자를 소문자로 바꾸거나 소문자를 대문자로 바꿀 방법이 없습니다.
그러면 다시 선공은 대문자를 하나 가져가고 나머지 쌍을 그대로 두기를 반복하면 됩니다.

이때 후공의 최선은 모든 쌍을 뒤집어서 `A b C d ... D c B`를 만드는 것입니다. 그러면 선공은 반드시 `A`를 가져가게 되고, $b \le B$, $c \le C$, ...가 이미 성립하기 때문에 선공이 추가로 이득을 볼 여지가 없어집니다.

따라서, 시작할 때 사과가 홀수 개라면, 루루는 사과의 값이 0이면서 나머지 사과들이 모두 대칭인 사과를 선택해야 점수를 같게 만들 수 있고 (즉 루루 승), 그 외의 경우 테라가 이깁니다.

시작할 때 사과가 짝수 개일 경우, 루루는 모든 사과 중에서 최솟값을 선택하여 테라에게 떠넘길 수 있습니다. 그러면 테라는 그 사과를 가져가야 하고,
테라가 남은 사과를 어떻게 섞든 루루는 홀수 개인 턴을 갖게 되기 때문에 루루는 반드시 적어도 테라만큼의 점수를 얻을 수 있습니다. 따라서 문제의 조건에 따라 루루가 이깁니다.
