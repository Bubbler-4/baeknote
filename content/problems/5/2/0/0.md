## 문제 내용

가로 길이 $w$, 세로 길이 $h$의 직사각형 모래판에 모래성을 쌓고 모래판의 안쪽 테두리를 따라 폭 $m$, 깊이 $d$인 해자(moat; 성을 방어하기 위해 성 밖을 파고 물을 채운 식의 함정)를 파려고 합니다.
각 모래성은 높이 $h_i$이고 밑면의 반지름이 $r_i$인 원기둥 모양입니다.

모래의 전체 양이 유지되도록 지면의 높이를 조절하려고 할 때, 지면의 높이의 변화량을 출력하세요.

### 입력

첫 줄에는 테스트 케이스의 개수 $K$가 주어집니다. ($K \ge 1$)

각 테스트 케이스에 대해, 첫 줄에는 $w$, $h$, $m$, $d$, 그리고 모래성의 개수 $b$가 주어집니다. ($0 \le b \le 100$, $m \le \frac{\min(w, h)}{2}$)

다음 $b$줄에는 $h_i$와 $r_i$가 주어집니다. 어떤 모래성의 밑면의 면적이 해자를 제외한 모래판의 면적을 초과하는 경우는 주어지지 않습니다.

### 출력

각 테스트 케이스에 대해, 먼저 `Data Set {테스트 케이스 번호}:`를 한 줄에 출력하고, 다음 줄에 문제의 정답을 소수점 이하 2자리로 반올림하여 출력합니다. 지면이 높아졌다면 양수로, 낮아졌다면 음수로 출력합니다.

## 문제 풀이

{{< details title="스포일러" closed="true" >}}

먼저, 모래성과 해자에 의해 변화된 모래의 양을 구합니다. 각 모래성의 부피는 $C = \sum h_i r_i^2 \pi$이고, 해자의 부피는 $M = (wh - (w-2m)(h-2m))d$입니다.

지면의 높이가 $x$만큼 높아지는 것은 모래판의 모든 점의 높이를 $x$만큼 높이는 것과 같으므로, 모래의 부피는 $whx$만큼 증가합니다. 반대로 $-x$만큼 낮아지면 모래의 부피는 $-whx$만큼 감소합니다.

따라서 답은 $\frac{M - C}{wh}$입니다.
